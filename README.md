# Fragment-Based Drug Discovery Pipeline

An automated AI-driven pipeline for designing novel drug-like molecules using fragment-based methods and 3D property optimization.

### Features
1. Upload active structures (SMILES/SDF)
2. Generate property-matched decoys
3. Train Random Forest classifiers on activity
4. Evaluate Fragment Libraries (BRICS)
5. Scaffold merging and Fragment linking
6. 3D Conformer generation and energy minimization (ETKDG)
7. Novelty checking against ChEMBL, PubChem, and Reaxys
